# Algoritmos e Lógica de Programação - C++

**Curso:** Engenharia de Computação  
**Instituição:** IFSP - Campus Birigui  
**Disciplina:** Algoritmos e Lógica de Programação  
**Linguagem:** C++

## Objetivo
Este repositório serve como apoio aos estudos da turma de Engenharia de Computação do IFSP Birigui. O foco é documentar e explicar as lógicas de resolução de atividades práticas desenvolvidas em C++.

## O que você encontrará aqui
- Soluções de exercícios propostos em aula.
- Comentários detalhados no código explicando o raciocínio lógico e algoritmos utilizados.
- Exemplos de estruturas de controle, vetores, funções e ponteiros.
